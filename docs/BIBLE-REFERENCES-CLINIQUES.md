## I. CORPUS DE RÉFÉRENCES

### Source 1 — NORSE Diagnostic Evaluation
- **Auteurs** : Gaspard N, Sculier C, Hirsch LJ, Hocker S, Sheikh Z, Ghoshal S, Pulluru Y et al.
- **Institution** : NORSE Institute Medical Advisory Board
- **Date** : Septembre 2020 (dernière mise à jour)
- **Type** : Protocole diagnostique de référence
- **Moteurs** : IntakeAnalyzer, TDE, EWE, PVE

**Contenu clé** :
- Évaluation dans les premières 24h : protocole SE institutionnel, histoire clinique, IRM cérébrale ± ARM/VRM, EEG continu vidéo
- 7 sections de diagnostic différentiel :
  - Section 1 : Bilan initial (sérologiques, PCR LCR, métagénomique)
  - Section 2 : Auto-immun/paranéoplasique (panel anticorps complet : LGI-1, CASPR2, NMDA, AMPA, GABA-B/A, GAD65, MOG, AQP4, D2R, IgLON5 + 20 autres)
  - Section 3 : Néoplasique (CT TAP, échographie, PET-CT)
  - Section 4 : Métabolique (BUN/Cr, ammoniaque, porphyrie, mitochondrial)
  - Section 5 : Toxicologique (benzodiazépines, amphétamines, métaux lourds)
  - Section 6 : Génétique (MERRF, MELAS, POLG1, SURF1, exome)
  - Section 7 : Dosage cytokines (IL-1β, IL-1Ra, IL-6, IL-10, IL-17, TNF-α, HMGB1, CXCL8-11)
- Évaluation à 48h et 72h : escalade immunothérapie si cryptogénique
- Table 2 : Facteurs géographiques/zoonotiques (Afrique, Australie, Asie, Europe, Amériques)
- Table 3 : Patterns neurologiques → diagnostic spécifique (Extreme Delta Brush → anti-NMDAR ; dyskinésies orofaciales → anti-NMDAR ; FBD → LGI1)
- Table 4 : Médicaments proconvulsivants (antibiotiques, antifongiques, chimiothérapies, immunosuppresseurs, CAR-T)
- Flowchart thérapeutique : L1 (corticoïdes/IVIG/plasmaphérèse) → L2 (rituximab/tocilizumab/anakinra/KD) → autres

### Source 2 — Yale New Haven Health SE Treatment Algorithm
- **Institution** : Yale New Haven Health — Dept. Pharmacy & Neurology
- **Date** : Juillet 2023 (dernière révision)
- **Type** : Algorithme thérapeutique (guideline)
- **Moteurs** : TDE, PVE

**Contenu clé** :
- 3 paliers temporels :
  - **<10 min** : Lorazepam 4mg IV (répéter ×1). Si pas d'IV : Diazepam 20mg PR ou Midazolam 10mg IN/buccal/IM
  - **10-30 min** : Levetiracetam 60mg/kg (max 4500mg) OU Valproate 40mg/kg (max 4000mg) OU Fosphenytoin 20mg PE/kg
  - **>30 min** : Kétamine 1.5mg/kg (push) OU Pentobarbital 5mg/kg
- Alternatives : Lacosamide 10mg/kg, Brivaracetam 6mg/kg, Phénobarbital 15mg/kg
- Perfusions continues : Midazolam, Propofol (max 48h à >80mcg/kg/min), Kétamine, Pentobarbital
- Médications entérales : Perampanel, Topiramate, Clobazam, Pregabaline, Oxcarbazépine, Vigabatrine
- Table pharmacocinétique complète : 15 molécules avec dosages, demi-vies, ajustements rénaux/hépatiques, interactions, commentaires cliniques
- Points critiques PVE :
  - Propofol : syndrome de perfusion (acidose, bradycardie, arrêt cardiaque, rhabdomyolyse)
  - Phénytoine : propylène glycol → acidose métabolique, extravasation → purple glove syndrome
  - Valproate : encéphalopathie hyperammoniémique, tératogène
  - Carbamazépine : HLA-B*1502 → Stevens-Johnson (population asiatique)
  - Kétamine : propriétés sympathomimétiques, possible ↑PIC

### Source 3 — Consensus International NORSE/FIRES (détaillé)
- **Auteurs** : Wickstrom R, Taraschenko O, Dilena R, Payne ET, Specchio N, Nabbout R, Koh S, Gaspard N, Hirsch LJ + International NORSE Consensus Group
- **Journal** : Epilepsia, Vol. 63, No. 11, pp. 2840-2864
- **Date** : Août 2022
- **DOI** : 10.1111/epi.17397
- **Type** : Consensus Delphi — 85 recommandations, 48 experts internationaux
- **Moteurs** : Tous (référence transversale)

**Contenu clé** :
- Méthode Delphi : 2 rounds, échelle Likert 1-9, consensus ≥7
- Panel : 48 experts (neurologie adulte/pédiatrique, épileptologie, neuro-réanimation, rhumatologie)
- 5 sections de recommandations :
  1. Caractéristiques de la maladie NORSE/FIRES
  2. Tests diagnostiques et échantillonnage
  3. Traitement phase aiguë
  4. Traitement phase post-aiguë
  5. Recherche, registres et directions futures
- Points critiques intégrés dans PULSAR :
  - NORSE et FIRES = même spectre, tous âges
  - FIRES = sous-catégorie de NORSE (maladie fébrile 2 semaines à 24h avant)
  - Pas d'essai contrôlé randomisé existant → décisions basées sur consensus expert
  - 2/3 des institutions US n'ont pas de protocole NORSE
  - Immunothérapie L1 recommandée dans les 72h si cryptogénique
  - Immunothérapie L2 + régime cétogène dans les 7 jours

### Source 5 — Biomarqueurs NORSE : SOPs et Biobanking
- **Auteurs** : Hanin A, Cespedes J, Pulluru Y, Gopaul M, Aronica E, Helbig I, Howe CL, Huttner A, Koh S, Navarro V, Taraschenko O, Vezzani A, Wilson MR, Xian J, Gaspard N, Hirsch LJ
- **Journal** : Epilepsia, 2023;64(6):1444-1457
- **DOI** : 10.1111/epi.17600
- **PMCID** : PMC10756682
- **Type** : Revue + SOPs pour biospecimens
- **Moteurs** : Discovery Engine (N1 Pattern Mining, N2 Literature Scanner)

**Contenu clé** :
- NORSE = 2ème urgence neurologique après l'AVC (SE : 40/100 000/an)
- ~25% des SE sont réfractaires (RSE)
- Mortalité NORSE : ~12% enfants, 16-27% adultes
- Majorité cryptogénique — pas de thérapie spécifique
- Biomarqueurs proposés : immunologiques, infectieux, génétiques
- Aucun biomarqueur validé à ce jour (variabilité des cohortes, méthodes, endpoints)
- NORSE Institute Biorepository : collection internationale de biospecimens
- SOPs standardisées : LCR, sérum, plasma, PBMC, urine, selles, tissu cérébral
- Critères d'utilisation appropriée des biospecimens préalablement collectés
- Implication PULSAR : le Pattern Mining (N1) et le Literature Scanner (N2) font exactement ce travail de corrélation que les SOPs tentent de standardiser — mais en temps réel, computationnellement

### Source 6 — Long-Term Outcomes NORSE
- **Auteurs** : Espino PH, Eschbach K, Blank LJ, Cervenka MC, Muscal E, Farias-Moeller R, Gilmore EJ, Gopaul MT, Haider HA, Hanin A, Hirsch LJ + 20 co-auteurs
- **Journal** : Epilepsia, 2025;66(4):988-1005
- **DOI** : 10.1111/epi.18267
- **PMCID** : PMC11997932
- **Type** : Consensus — outcomes long-terme (NORSE Institute LTO Working Group)
- **Moteurs** : TPE (prospection), Discovery N3 (hypothèses)

**Contenu clé** :
- 8 domaines d'outcomes long-terme :
  1. **Fonctionnement** : Neuropsychologique, Neurologique (hors crises), Psychiatrique (humeur, comportement)
  2. **Qualité de vie**
  3. **Épilepsie**
  4. **Médical** (non-neurologique)
  5. **Social**
  6. **Charge des aidants**
  7. **Mortalité long-terme**
  8. **Impact système de santé**
- Timing d'évaluation : 6 mois, 1 an après sortie, puis annuellement
- Mortalité intra-hospitalière : 10-42% (variable selon étiologie, âge, durée RSE, comorbidités)
- Hétérogénéité massive dans les méthodes d'évaluation actuelles
- Implication PULSAR : le TPE peut intégrer ces 8 domaines dans ses projections J+7/J+14

### Source 7 — Rapport Technique Neuro-inflammation Pédiatrique
- **Auteur** : Document de travail interne
- **Date** : 2026
- **Type** : Synthèse structurée avec données chiffrées et sources
- **Moteurs** : TDE, PVE, Discovery N4 (Treatment Pathfinder)

**Contenu clé** :
- Distinction FIRES/NORSE (inflammation innée) vs EAIS (mécanisme adaptatif/anticorps)
- Tableau d'efficacité thérapeutique :
  - FIRES : Anakinra + KD → ~75% contrôle crises
  - EAIS (NMDA) : Corticoïdes + IgIV → ~80%
  - EAIS (séronégative) : Corticoïdes + Rituximab rapide → ~60%
- Physiopathologie : tempête de cytokines (IL-1β >10× norme, IL-6, CXCL10/IP-10)
- EEG : Extreme Delta Brush dans 30% NMDAR/FIRES
- IRM : Hypersignaux T2/FLAIR limbiques dans 50% EAIS
- Burst Suppression >48h → ↑35% risque atrophie cérébrale
- Anakinra : réponse ~60-65% FIRES, réduit crises 50% en <7j si débuté avant J-14
- Régime cétogène : seuil BHB >4.0 mmol/L pour arrêt crises (>75% réduction)
- Protocole couplé 2026 : Anakinra + KD ratio 4:1 → 85% sevrage anesthésiques (vs 40% sans)
- Risques Anakinra : infections sévères 20-25%, interaction corticoïdes +40% risque infectieux
- Pronostic FIRES : mortalité 10-30%, séquelles cognitives QIT<70 dans 70-90% survivants
